عن رحّال About Rahhal

رحّال هي منصة سياحية ذكية داخل المملكة العربية السعودية تجمع تخطيط الرحلات واستكشاف الفعاليات وإدارة الرحلات في نظام واحد. تساعد المستخدمين على اكتشاف الوجهات والفعاليات، إنشاء خطط سفر مخصصة بالذكاء الاصطناعي، وإدارة الرحلات الجماعية والمدرسية بسهولة. Rahhal is a smart tourism platform in Saudi Arabia that combines trip planning, event discovery, and trip management in one system. It helps users discover destinations and events, create personalized AI itineraries, and manage group and school trips with ease.

1 اكتشاف الوجهات والفعاليات السياحية من نفس المنصة Discover destinations and tourism events from the same platform
2 خطة سفر مخصصة حسب المدينة والميزانية والاهتمامات Create tailored travel plans by city, budget, and interests
3 إدارة الرحلات الجماعية والمدرسية والحجوزات بسهولة Manage group trips, school trips, and bookings with ease

المشكلة Problem Statement

تعتمد معظم الخدمات السياحية الحالية على منصات منفصلة للبحث عن الفعاليات، التخطيط للرحلات، وإدارة الحجوزات، مما يسبب تجربة مشتتة للمستخدم. كما تفتقر الكثير من الحلول إلى التخطيط الذكي للرحلات ودعم الرحلات المدرسية وإدارة المجموعات بشكل متكامل. Most current tourism services rely on separate platforms for event search, trip planning, and booking management, creating a fragmented user experience. Many solutions also lack intelligent trip planning and integrated support for school trips and group management.

المزايا الأساسية داخل تجربة رحّال The core capabilities inside Rahhal

تخطيط رحلة بالذكاء الاصطناعي AI Trip Planner

إنشاء خطة سفر مخصصة بناءً على الوجهة والميزانية والاهتمامات. Generate a personalized travel plan based on destination, budget, and interests.

اكتشاف الفعاليات Event Discovery

استعراض الفعاليات السياحية والانتقال لمصادر الحجز الرسمية. Browse tourism events and move to official booking sources.

الرحلات الجماعية Group Trips

إنشاء رحلات جماعية أو الانضمام إلى رحلات منشأة من قبل مستخدمين آخرين. Create group trips or join trips created by other users.

الرحلات المدرسية School Trips

إدارة الرحلات المدرسية وموافقات أولياء الأمور والتقارير. Manage school trips, parent approvals, and reports.

حجز QR QR Booking

عرض الحجوزات والوصول إليها باستخدام QR Code. Access bookings using QR codes.

تقنيات رحّال والأنظمة الداعمة Rahhal's technologies and supporting systems

Frontend Frontend 01

React.js

يبني تجربة استخدام مرنة وسريعة لعرض الوجهات والفعاليات والتخطيط. Builds a fast, flexible interface for destinations, events, and planning.

Backend Backend 02

NestJS

ينظم الخدمات الخلفية الخاصة بالرحلات والفعاليات والحجوزات والموافقات. Organizes backend services for trips, events, bookings, and approvals.

Database Database 03

MySQL

يوفر تخزينًا منظمًا للبيانات الخاصة بالمستخدمين والرحلات والوجهات والحجوزات. Provides structured storage for users, trips, destinations, and bookings.

AI 04

OpenAI API

يدعم إنشاء خطط سفر ذكية مخصصة بحسب معطيات المستخدم. Supports personalized AI itinerary generation.

Email 05

SendGrid

يرسل التنبيهات والمراسلات المتعلقة بالموافقات والحجوزات. Handles notifications for approvals and bookings.

Integration 06

REST APIs

تربط المنصة مع مصادر البيانات والخدمات الخارجية بشكل قابل للتوسع. Connects the platform with external services and data sources.

Workflow 07

GitHub

يدعم إدارة الكود والتعاون بين أعضاء الفريق ومتابعة التطوير. Supports code management, collaboration, and development tracking.
